Steps to run Parameter Generation Tool


================
For help on tool
================
/p/com/eda/intel/siputils/prod/bin/configIP -h
        OR
  In GUI, press the global help to get the pdf version


================
For running the tool
================
/p/com/eda/intel/siputils/prod/bin/configIP  dfxsecure_plugin.config -odf <ip_name>_input_file.vh


================
To run the tool with prepopulated values and save it back to the smae file
================
/p/com/eda/intel/siputils/prod/bin/configIP  dfxsecure_plugin.config -odf <ip_name>_input_file.vh -if <ip_name>_input_file.vh
